# Civic Tech Glossary

Terms and concepts you'll encounter at Civic Tech Montreal.

## Core Concepts

### Civic Technology
Technology developed to improve government and citizen services, increase civic participation, and solve public problems.

### Open Data
Public information made freely available in machine-readable formats, enabling transparency and innovation.

### Civic Engagement
Participation by residents in democratic processes, community organizations, and public decision-making.

### Government Data
Information collected and maintained by government agencies, often available through data portals and open data initiatives.

### Accessible Technology
Software and services designed to be usable by people with disabilities, including visual, hearing, and mobility impairments.

## Technical Terms

### API (Application Programming Interface)
A set of rules allowing different software applications to communicate with each other.

### GitHub
A platform for version control and collaboration on code projects.

### Database
Organized collection of data that can be accessed and updated.

### Frontend
The user interface and client-side code of an application.

### Backend
Server-side code and databases that power an application.

## Civic Tech Concepts

### E-Government
Use of technology to deliver government services and enable citizen participation online.

### Smart Cities
Use of data and technology to improve urban services and citizen quality of life.

### Democratic Participation
Active involvement of citizens in governance, voting, and public decisions.

### Community Problem-Solving
Using technology and collaboration to address local community challenges.
