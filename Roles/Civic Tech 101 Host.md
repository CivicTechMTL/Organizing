# Civic Tech 101 Host

**Commitment length:** 1 Month

## Role Description

The Civic Tech 101 Host introduces new members to the world of civic technology in Montreal. They facilitate a dedicated project room for this purpose, using a prepared presentation and fostering discussion to engage newcomers.

## Responsibilities

1. **Prepare the Room** — Set up the designated project room for the Civic Tech 101 session.

2. **Welcome Newcomers** — Greet attendees who join the Civic Tech 101 room with a warm welcome.

3. **Initiate Discussion** — Start a conversation about civic technology by:
   - Asking attendees about their understanding of civic tech
   - Asking about their interests
   - Encouraging brainstorming and idea sharing

4. **Present Civic Tech 101 Presentation** — Use the provided PowerPoint presentation to:
   - Introduce key concepts related to civic tech in Montreal
   - Share relevant resources
   - Encourage questions and discussion throughout

5. **Facilitate Interaction** — Create a friendly and inclusive environment by:
   - Helping attendees connect with each other
   - Introducing attendees to existing projects
   - Maintaining open communication

6. **Answer Questions** — Be prepared to answer questions about:
   - Civic technology and its applications
   - The community and how to stay involved
   - Potential projects to join
   
   If you don't know the answer, direct the question to the organizers.

7. **Encourage Participation** — Invite attendees to:
   - Participate in future meetups
   - Join active projects
   - Stay connected with the community

8. **Wrap Up the Session** — Thank everyone for attending and let them know where they can find more information about the community.
