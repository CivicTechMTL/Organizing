# Host (Emcee)

**Commitment length:** 1 Month

## Role Description

The Host is the emcee for the weekly civic tech meetups, responsible for creating a positive and inclusive space. They guide the flow of the event, facilitate introductions, manage project callouts, and ensure a smooth and engaging experience for all attendees.

## Pre-Event Setup

Before the meetup, review and update the presentation deck with the following:
- Next event date
- Speaker information
- Projects page

## Responsibilities

1. **Start on Time** — Begin the meetup at the scheduled start time.

2. **Facilitate Introductions** — Ask attendees to introduce themselves by sharing:
   - Their name
   - Their pronouns
   - Something they are grateful for
   
   Encourage a welcoming and inclusive atmosphere.

3. **Speaker Introduction** *(if applicable)*
   - Introduce the speaker and their topic
   - Give the space to the speaker

4. **Q&A Session** *(if applicable)*
   - Moderate the Q&A to keep it on topic
   - Ensure everyone has a chance to ask questions
   - Finish with time for one hour of project work

5. **Project Callouts** — Ask if anyone is working on a project and would like to share. For each project, ask:
   - What is the project?
   - What skills are needed?
   - Request a short description of the project
   
   Be sure to include:
   - An open brainstorm project callout
   - The civic tech 101 session callout

6. **Check In on Project Teams** — Before wrapping up, ask if any project teams need to share updates or require additional support.

7. **Wrap Up on Time**
   - Thank attendees for their participation
   - Announce any upcoming events or announcements
   - End the meetup at the scheduled end time
