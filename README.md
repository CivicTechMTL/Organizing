# Civic Tech Montreal — Runsheets

This folder contains role descriptions and responsibilities for volunteers at Civic Tech Montreal meetups.

## Roles

### Event Facilitation Roles

- **[Greeter](./greeter.md)** — 1 Month  
  First point of contact for attendees. Welcomes people and guides them to the event space.

- **[Host (Emcee)](./host.md)** — 1 Month  
  Guides the flow of the meetup, facilitates introductions, and manages the event agenda.

- **[Civic Tech 101 Host](./civic-tech-101-host.md)** — 1 Month  
  Leads a dedicated session introducing newcomers to civic technology concepts and projects.

### Project Leadership

- **[Project Lead](./project-lead.md)** — Variable  
  Guides and manages a specific civic tech project, fostering collaboration and progress.

### Logistics & Outreach

- **[Venue Finder](./venue-finder.md)** — 3 Months  
  Secures suitable venues for weekly meetups with necessary facilities and accessibility.

- **[Speaker Finder](./speaker-finder.md)** — 3 Months  
  Identifies and books speakers for meetup presentations on civic tech topics.

---

## Using These Runsheets

Each role file includes:
- A brief description of the role
- Expected commitment length
- Detailed, step-by-step responsibilities
- Email templates or other resources where applicable

Select a role to see the full details and responsibilities.
